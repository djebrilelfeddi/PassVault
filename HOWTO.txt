La documentation est dans \target\reports\apidocs\index.html
L'executable est dans \target\PasswordManager-1.0-SNAPSHOT\bin\PasswordManager (crée avec  mvn clean package javafx:jlink)
OU si jamais vous avez un pb vous pouvez build et run via Netbeans.

Les fichiers de données (mdp, config utilisateur) sont stocké dans \target\PasswordManager-1.0-SNAPSHOT\bin\users_data\.

Le fichier de config sera du type <username>_config.txt
Le fichier de mdp est du type <username>_password.txt

Le fichier de mdp est encrypté en AES-256-GCM, et une fois decrypté, chaque mot de passe est encrypté avec l'encodage specifique de l'utilisateur (Algorithme+Mode choisi a l'inscription)

Interface avec SceneBuilder, un peu de CSS.

Quelques fonctionnalités en plus. (Une MessageBox avant que ça se ferme!)

Enjoy ! : )
